## About

The Lattice Lock Framework provides:
- Architecture rule enforcement via `lattice.yaml`
- Static analysis (Sheriff) for catching violations
- Runtime testing (Gauntlet) for semantic properties
- Model orchestration for AI-powered development
- Comprehensive governance workflow

## Key Standards

All file names must be **lowercase with underscores** (e.g., `my_file.py`, not `MyFile.py`).

For complete details, see [contributing.md](contributing.md).
